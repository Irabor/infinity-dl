# infinity-dl
### Description
ifinity-dl downloads all images or videos on chan threads and saves them in a specified directory
```
infinity-dl.py [URL] [DIR]
```

Supported chan:

* 8ch
* 4chan
* enchan
* 7chan
* hispanchan
* 16chan
